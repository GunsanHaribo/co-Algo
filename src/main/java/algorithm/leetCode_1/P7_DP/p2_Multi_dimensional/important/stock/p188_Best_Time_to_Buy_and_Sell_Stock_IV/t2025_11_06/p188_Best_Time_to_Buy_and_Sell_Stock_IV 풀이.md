1. 문제를 선택 문제(여러 케이스로 결정)
- 주식 없음 -> 살기(buy), 스킵
- 주식 있음 -> 팔기(sell), 스킵
2. 선택 문제로 봤다면 -> 의사결정에 영향을 주는 거 파악
- 오늘 사도 되는지 알려면?, 뭘 알고 있어야되나
3. 상태 전이(케이스)
- 주식 없음(holding=0) + buy → holding=1, remain 변화 없음
- 주식 있음(holding=1) + sell → holding=0, remain-1
4. 오늘의 최대 이익 = 오늘 행동 + 내일의 최대 이익 미래 기반 dp정의(현재 → 미래)
- 오늘 산다 → 내일부터(i+1) 주식을 갖고 있는 상태가 된다.
- 오늘 안 산다 → 내일부터(i+1) 여전히 미보유 상태이다.
- 오늘의 최대 이익 = 오늘 행동 + 내일의 최대 이익  -> dp[i] = ... + dp[i+1]
