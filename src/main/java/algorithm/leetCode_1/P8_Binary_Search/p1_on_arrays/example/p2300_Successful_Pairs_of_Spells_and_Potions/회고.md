-> 정답 후보 범위 찾기
- “mid를 버릴지 살릴지” 에 따라 mid, mid±1 를 쓴다

- left<=right, mid = right-1 -> 정확한 값 찾기(search for exact element)
- left<right, mid = right -> 하한/상한(lower/upper bound) 찾기, 삽입 위치 구하기
  - left == right → 후보가 없다 → 탐색 끝
  - right는 '가능한 정답의 바로 오른쪽'
  - [left, right) 안에는 경계가 있을 수 있다.
    [0, left)     = 반드시 a[i] < target
    [right, n)    = 반드시 a[i] ≥ target
