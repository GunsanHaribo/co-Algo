    - 가능한 합을 전달해야할까, 너무 많은데
    - 왼쪽과 오른쪽이 -> 루트를 뺸값 하고 같은지 확인 
    - 하나라도 있으면 T -> OR연산 사용
2. 목표합인지 비교

    - 합해서 목표합인지 확인
