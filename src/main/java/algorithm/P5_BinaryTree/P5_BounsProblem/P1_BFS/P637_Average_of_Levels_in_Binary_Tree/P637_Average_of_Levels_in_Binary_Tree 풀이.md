# 문제 
~~~text
이진 트리의 루트 노드가 주어졌을 때, 각 레벨의 노드들의 평균값을 배열의 형태로 반환하시오.
정답은 실제 값과의 오차가 10⁻⁵ 이내이면 허용됩니다.
~~~

# 분석
1. 미지 : 평균값(각 레벨의 노드들 )
2. 자료 : 이진 트리의 루트노드
3. 조건 :
- 평균값을 배열 형태로 반환 

# 풀이 
~~~text
이진 트리의 루트 노드가 주어졌을 때, 각 레벨의 노드들의 평균값을 배열의 형태로 반환하시오.
정답은 실제 값과의 오차가 10⁻⁵ 이내이면 허용됩니다.
~~~

- BFS에 레벨별로 average 계산
